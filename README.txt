PAINT is the curational interface for GO annotation for protein families.

## Users

To run paint, get the latest version from

https://github.com/geneontology/paint/releases

Download paint-all.jar and the launchPAINT script (either `.sh` or `.bat` depending on whether you are unix/Mac or windows). Make sure these are in the same directory. Run the script.

## Developers

PAINT is dependent on owltools and touchup

Once installed and you've done what you want build with
mvn clean install
